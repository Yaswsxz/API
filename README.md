# 🌸 Studio Ghibli API

Um site inspirado no universo encantador do Studio Ghibli desenvolvido com HTML, CSS e JavaScript, consumindo uma API pública para exibir personagens, filmes e informações de forma dinâmica.

## ✨ Funcionalidades

- 🎬 Listagem de filmes do Studio Ghibli
- 👤 Exibição de personagens
- 🔎 Sistema de busca
- 📱 Layout responsivo
- ⚡ Consumo de API em tempo real
- 🎨 Interface inspirada na estética Ghibli

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- API pública Studio Ghibli
